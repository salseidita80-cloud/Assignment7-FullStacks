U.S. President SCRUD App

Assignment 7 — Full stack application with a Streamlit frontend consuming a FastAPI + Supabase RESTful backend.

What the app does

This is a full stack web application that lets users perform all five SCRUD operations on a database of U.S. Presidents. The Streamlit frontend communicates with a FastAPI backend deployed on Render, which connects to a Supabase PostgreSQL database.

Tech stack

Python Streamlit FastAPI Supabase RESTful API Render GitHub Pages

SCRUD functions

  • Search — Filter all presidents by first or last name in real time
  • Create — Add a new president with first name, last name, and birthdate
  • Retrieve — Fetch a specific president by their numeric ID
  • Update — Edit a president's details by ID
  • Delete — Remove a president record by ID

How the frontend consumes the backend

The Streamlit app uses Python's requests library to call the RESTful API endpoints hosted at https://myapi1-pz44.onrender.com. Each SCRUD operation maps to a specific HTTP method and endpoint:


GET /presidents — Load all presidents (Search)
GET /presidents/{id} — Retrieve one president by ID
POST /presidents — Create a new president
PATCH /presidents/{id} — Update a president's details
DELETE /presidents/{id} — Delete a president

How to run locally

  1. Clone the repo: git clone https://github.com/salseidita80-cloud/Assignment7-FullStacks
  2. Install dependencies: pip install -r requirements.txt
  3. Run the app: streamlit run app.py
